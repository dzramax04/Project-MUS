# Project-MUS

**Monetary Unit Sampling (MUS) Tool untuk Profesional Audit**

Project-MUS adalah aplikasi web berbasis HTML, CSS, dan JavaScript yang dirancang khusus untuk membantu auditor dalam melakukan **sampling berbasis nilai (Monetary Unit Sampling)** secara efisien dan akurat, sesuai dengan standar audit profesional terkini.
